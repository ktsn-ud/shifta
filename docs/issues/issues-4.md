# [issues-4] 一括同期ボタンの実装

## 概要

ダッシュボード（/my） のカレンダーの上部あたりに高さ 3em 程度のパネルを設置し、全体の同期状態を表示する。同期状態は適切なアイコンとメッセージで表現する。

- [すべてのシフトが正常に同期できている場合] Google Calendar に正常に同期できている旨のメッセージを表示
- [正常に同期できていない場合] ○件のシフトが Google Calendar に同期できていないというメッセージを表示し、その横に「一括して再同期」のボタンを設置する。

一括して再同期のボタンを押すと、同期できていないシフトをまとめて再試行する。 sonner に成功件数と失敗件数の結果を表示し、同期状態を更新する。
