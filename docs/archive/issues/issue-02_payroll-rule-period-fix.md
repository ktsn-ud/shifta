# [issues-2] 給与ルールの適用期間に関する修正

## 概要

以下のそれぞれについて、コミットを分けて実装を行う。

- fix: 表示上の終了日は適用される最後の日付を表す。
  - 例えば、「〜 2026-03-31」と表示される場合、それは「2026-04-01 0:00」の直前までである。つまり、UI上で終了日を 2026-03-31 とした場合は、 PayrollRule.endDate を 2026-04-01 として登録する。
  - lib/payroll/summarizeByPeriod.ts の findApplicablePayrollRule は正しいのでそのままでよい。
- feat: 既に「〜現在」の給与ルールがある状態で、その期間に重複するような新たな給与ルールを設けた場合、古い方の給与ルールの終了日を新たな給与ルールの開始日の前日に更新する。
