# Archive Docs

`docs/` 直下には現行で参照する主要ドキュメントのみを残し、過去の計画・調査・課題は本ディレクトリに集約しています。

## フォルダ構成

- `performance/`: パフォーマンス調査・最適化計画・監査
- `google-calendar/`: Google Calendar 連携の実装計画
- `feature-plans/`: 機能仕様案・実装計画（履歴）
- `issues/`: 過去 issue 起点の実装依頼メモ
- `implementation/`: 初期実装タスク分割書などの旧計画資料

## 命名規則

- 日付が明確な資料: `YYYY-MM-DD_<topic>.md`
- issue メモ: `issue-XX_<topic>.md`
- 日付不明の旧資料: `legacy_<topic>.md`

この規則により、同一ジャンル内で時系列に並ぶようにしています。
